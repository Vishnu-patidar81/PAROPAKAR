# Blood Donation Web Application

A web-based Blood Donation Management System developed using Java Servlets, JSP, and SQL. The application helps connect blood donors and recipients efficiently by providing donor registration, blood search, and request management functionalities.

## Features

* Donor Registration and Login
* Search Blood Donors by City and Blood Group
* Blood Request Management
* Store and Manage Donor Details
* Responsive User Interface
* Database Connectivity using JDBC

## Technologies Used

* Java
* Servlets
* JSP
* JDBC
* SQL
* HTML
* CSS

## Project Objective

The main objective of this project is to provide a platform where users can quickly find blood donors during emergency situations and improve accessibility to blood donation services.

## How to Run

1. Import the project into Apache
2. Configure Apache Tomcat Server
3. Create the SQL database and import required tables
4. Update JDBC database credentials
5. Run the project on the server
